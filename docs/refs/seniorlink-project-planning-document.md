# SeniorTF (시니어링크) 프로젝트 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트명

**SeniorTF (시니어링크)** — 경험의 재순환, 시니어 전문성과 단기 TF를 잇는 AI 매칭 플랫폼

### 1.2 핵심 개념

퇴직 시니어 전문가와 단기 TF가 필요한 기업을 AI로 정밀 매칭하는 커뮤니티 플랫폼입니다. 시니어의 경력과 전문성을 단기 프로젝트에 재순환시키며, 계약과 정산까지 앱 내에서 통합 관리합니다.

### 1.3 프로젝트 기간

**15개월** (Phase 0~3, 2026년 M1~M15)

---

## 2. 사업 배경 및 필요성

### 2.1 시장 현황

- 한국은 2025년 초고령사회(65세 이상 20% 초과) 진입
- 베이비부머 세대 약 730만 명 현재 퇴직 중
- 퇴직자의 경력 활용률은 30%에 불과
- 국내 긱 이코노미 시장 연 15% 성장 중
- 시니어 전문가 특화 단기 매칭 플랫폼 사실상 부재

### 2.2 문제 정의

#### 공급자 (시니어 전문가)의 문제

- 법정 정년(60세)과 실질 은퇴(49~51세) 사이 최대 10년 경력 공백
- 기존 취업 플랫폼: 장기 정규직 중심, 시니어 단기 채널 부재
- 기회: 인맥 의존 → 지역·분야·직급별 극도의 불균등
- 결과: 경력 활용 부재 → 사회적 고립 → 우울·건강 악화

#### 수요자 (기업/기관)의 문제

- 검증된 시니어 전문가를 단기(3~6개월)로 구하는 채널 없음
- 헤드헌팅: 수수료 20~30% (단기 TF에 구조적으로 부적합)
- 직접 섭외: 네트워크 한계, 계약·세무 처리 번거로움
- 결과: 전문성 조달 실패 → 프로젝트 지연·실패율 상승

#### 기존 솔루션의 한계

| 기존 수단        | 한계                                                      |
| ---------------- | --------------------------------------------------------- |
| 사람인·잡코리아  | 장기 정규직 중심, 시니어 단기직 카테고리 부재             |
| 크몽·숨고        | 젊은 디지털 직군 위주, 고경력 오프라인 전문가 활용 어려움 |
| 정부 신중년 사업 | 오프라인 창구 중심, AI 매칭 없음, 기업 수요 연결 미흡     |
| 헤드헌팅         | 비용 과다, 단기 TF에 부적합한 구조                        |

### 2.3 해결 방식

1. 시니어 전문가는 경력, 전문 분야, 가용 기간, 희망 지역을 등록
2. 기업은 TF 목적, 기간, 예산, 필요 분야를 입력
3. AI가 프로젝트 특성에 맞춘 전문가 후보와 조합을 추천
4. 제안-수락-계약-정산까지 플랫폼에서 일관되게 처리

## 3. 서비스 개요

### 3.1 핵심 가치 제안

- **경험의 재순환**: 퇴직 전문성을 단기 TF로 재활용
- **신뢰형 매칭**: 고경력 시니어와 기업의 니즈를 정밀 연결
- **절차 간소화**: 계약서 자동 생성, 에스크로 정산, 진행 모니터링 일체화

### 3.2 주요 사용자

- **공급자**: 40~70대 퇴직 시니어, 대기업·공기업 출신 고경력자
- **수요자**: 스타트업, 중소기업, 공공기관 등 단기 TF가 필요한 기관

### 3.3 핵심 기능

1. **시니어 프로필 등록**: 전문 분야, 경력 기간, 가용 기간, 지역 입력
2. **기업 TF 요청**: 프로젝트 목적, 기간, 예산, 필요 분야 입력
3. **AI 매칭 추천**: 프로젝트 특성에 맞춘 전문가 후보 제시
4. **제안/수락 플로우**: 기업-시니어 간 제안 및 응답 처리
5. **계약서 자동 생성**: 수락 시 기본 계약 템플릿 자동 생성
6. **진행 및 정산**: 프로젝트 상태 관리, 에스크로 기반 정산

### 3.4 사용자 경험 흐름

#### 시니어 여정

1. 가입 및 프로필 작성
2. 매칭 제안 수신 및 확인
3. 제안 수락/거절
4. 계약 서명 및 프로젝트 시작
5. 진행 상태 관리 및 정산 요청
6. 완료 후 리뷰 작성

#### 기업 여정

1. TF 요청 작성
2. AI 추천 전문가 확인
3. 적합한 전문가에게 제안 발송
4. 수락 관리 및 계약 진행
5. 프로젝트 모니터링
6. 정산 처리 및 평가

## 4. 시장 분석

### 4.1 타겟 시장

- **지역**: 한국 전역 (서울 강남구 집중)
- **대상**: 40~70대 시니어 + 신규 사업 중인 기업
- **규모**: 초기 1년 5,000명 시니어 + 500개사 기업 목표

### 4.2 시장 진입 전략

1. **초기 베타 (3개월)**: 퇴직 네트워크 200명 + 파트너 기업 50개사
2. **AI 고도화**: 베타 데이터 기반 매칭 정확도 개선
3. **B2B 파트너십**: 중소기업 지원 기관, 창업지원센터, 산업협회 협업
4. **사용자 확장**: 시니어 커뮤니티, 동창회, 공공기관 채널 활용

### 4.3 경쟁 우위

- 시니어 특화 매칭 (기존 프리랜서 플랫폼과 차별화)
- 단기 TF 전용 플로우 (헤드헌팅 대비 낮은 비용)
- 계약·정산 일체화 (진입 장벽 최소화)
- 경험 기반 추천 + 신뢰 요소 (배지, 리뷰, 인증)

## 6. 기술 아키텍처

### 6.1 기술 스택

- **프론트엔드**: React Web + React Native
- **백엔드**: Node.js + Express/NestJS
- **데이터베이스**: PostgreSQL
- **인증**: JWT 기반
- **클라우드**: AWS/GCP
- **배포**: Docker + CI/CD (GitHub Actions)

### 6.2 주요 시스템 컴포넌트

- **API 서버**: 인증, 프로필, TF 요청, 제안, 계약, 정산
- **UI**: 시니어/기업 대시보드, 매칭 화면, 제안 수신함, 진행 관리
- **데이터베이스**: 사용자, 프로필, 요청, 제안, 계약, 정산, 리뷰
- **매칭 엔진**: 룰 기반 매칭 + 키워드 유사도 점수
- **문서 생성**: 계약서 템플릿 자동 생성

### 6.3 매칭 로직

1. **필터**: 기간, 예산, 지역, 전문 분야 일치 여부로 후보군 축소
2. **점수화**: 경력, 기술 키워드, TF 목적 일치도, 과거 프로젝트 유사도
3. **정렬**: 적합도 점수, 가용 기간, 우선순위로 추천

### 6.4 데이터 모델

- **시니어 전문가**: 전문 분야, 경력, 기술 키워드, 가용 기간, 지역, 상태
- **TF 요청**: 프로젝트명, 필요 분야, 팀 규모, 기간, 예산, 목표, 상태
- **제안**: 요청, 시니어, 상태, 메시지, 적합도 점수
- **계약**: 제안, 기간, 보수, 정산 상태, 진행률
- **정산**: 계약, 금액, 상태, 요청·완료 일시
- **리뷰**: 계약, 평점, 코멘트

## 7. 개발 계획

### 7.1 전체 타임라인


### 7.2 Phase 0 — 시장 검증 (M1~M2)

**목표**: 수요 실증 및 핵심 가설 검증

- 퇴직 전문가 30명 이상 심층 인터뷰
- 스타트업·중소기업 20개사 인터뷰
- 국내외 유사 플랫폼 벤치마킹 (탤런트뱅크, 크몽, YourEncore)
- 플랫폼 노동 관련 법률 검토
- 코어팀 구성

**성공 지표**: 사용 의향 70%+, 파트너 기업 3개사 확보

### 7.3 Phase 1 — MVP 개발 (M3~M8)

**목표**: 핵심 기능을 담은 작동 가능한 앱 완성

| 기간  | 개발 내용                                             |
| ----- | ----------------------------------------------------- |
| M3~M4 | 회원가입·프로필 (전문가/기업 구분), 경력 태그 시스템  |
| M5~M6 | AI 매칭 알고리즘 v1 (스킬 키워드·지역·가용 기간 기반) |
| M7    | TF 구성 요청·제안·수락 플로우, 채팅 기능              |
| M8    | 계약서 자동 생성, 에스크로 정산, QA·버그 수정         |

**기술 스택**: React Native(iOS·Android), Node.js + PostgreSQL, 토스페이먼츠 정산 연동

### 7.4 Phase 2 — 베타 테스트 (M9~M11)

**목표**: 실사용 데이터 수집, 매칭 품질 검증

- 클로즈드 베타: 전문가 200명, 기업 50개사
- 시나리오 실행: 프로필 등록 → 매칭 → 제안 → 계약 → 정산
- 매칭 수락률·완료율·양측 NPS 측정
- 베타 데이터 기반 AI 모델 재학습 및 고도화
- 분쟁 조정 프로세스, CS 응대 기준 수립

**성공 지표**: 매칭 완료 100건, 프로젝트 완료율 75%+, NPS 40점+

### 7.5 Phase 3 — 정식 출시 (M12~M15)

**목표**: 유료화 시작, 회원 확장, 수익 구조 안정화

- Google Play·App Store 정식 등록
- 기업 측 매칭 수수료(10~15%), 전문가 프리미엄 구독
- 중소기업진흥공단·창업진흥원 B2B 파트너십 협의
- 시니어 커뮤니티·경총·동창회 네트워크 활용 마케팅

**1차 목표**: 전문가 5,000명, 기업 500개사, 월 매칭 200건

## 12. 투자 포인트

### 12.1 시장 기회

- 초고령사회 진입으로 시니어 전문인력 활용 수요 급증
- 단기 TF 플랫폼 미개척 시장
- 공급(퇴직 전문가)과 수요(기업)가 동시에 명확

### 12.2 비즈니스 타당성

- 사회적 임팩트 + 비즈니스 가치 동시 확보
- 초기 네트워크 기반 시장 검증 가능
- 단계적 수익화 경로 명확 (수수료, 프리미엄, 정산 수수료)

### 12.3 팀 역량

- 창업자의 대기업·공기업 경험과 퇴직자 네트워크
- 기술·운영·비즈니스를 아우르는 코어팀 구성 계획
- 현실적이고 검증된 15개월 로드맵
