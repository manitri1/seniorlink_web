## Q2. 아이디어를 떠올린 배경 이야기

저는 대기업·공기업에서 수십 년을 일했습니다. 퇴직하던 날, 사무실 짐을 싸면서 이런 생각이 들었습니다. "이 경험들, 이제 어디에 쓰지?"

퇴직 후 가장 먼저 실감한 것은 '채널의 부재'였습니다. 취업 사이트를 열어보면 40대 이상을 위한 자리는 경비직, 단순 관리직이 대부분이었습니다. 제가 20~30년간 쌓아온 기획력, 조직 운영 경험, 산업 네트워크를 살릴 수 있는 단기 프로젝트 채널은 어디에도 없었습니다.

주변을 보면 같은 처지의 동료들이 많았습니다. 공기업 본부장 출신이 지인 소개로 겨우 자문 자리를 얻거나, 대기업 임원 경력자가 유튜브 강의로 방향을 돌리는 모습을 보며 안타까움을 느꼈습니다.

반대로 제 지인 중 스타트업을 운영하는 후배는 "팀장급 경험자를 3개월만 구할 방법이 없다"고 했습니다. 수요는 있는데 공급이 닿지 않는 것입니다.

이 두 현실이 겹쳐 보이는 순간, 시니어링크 아이디어가 시작되었습니다. 수십 년의 경험이 사장되는 것이 아니라, 필요한 곳에 흐를 수 있도록 하고 싶었습니다.

**시장 배경**

한국은 2025년 초고령사회(65세 이상 20% 초과)에 공식 진입했습니다. 베이비부머 세대 약 730만 명이 지금도 퇴직 중이며, 퇴직자의 경력 활용률은 30%에 불과합니다. 국내 긱(Gig) 이코노미 시장은 연 15% 성장 중이지만, 고경력 시니어 전문가에 특화된 단기 매칭 플랫폼은 존재하지 않습니다.

## Q3. 아이디어는 누구의 어떤 문제를 해결해주나요?

시니어링크는 두 그룹의 문제를 동시에 해결합니다.

### 공급자 — 퇴직 시니어 전문가 (40~70대)

저처럼 대기업·공기업을 떠난 분들은 이른 나이(평균 49~51세)에 사실상 은퇴합니다. 법정 정년(60세)까지 최장 10년, 그 사이 경력을 살릴 단기 채널이 없습니다. 기존 플랫폼은 장기 정규직 중심이고, 시니어 프리랜서 시장은 사실상 미개척 상태입니다. 결국 기회는 인맥에 달려 있어 지역·분야·직급에 따라 극도로 불균등합니다.

- 법정 정년(60세)과 실질 은퇴(49~51세) 사이 최대 10년 경력 공백 발생
- 기존 취업 플랫폼은 장기 정규직 중심 — 시니어 단기 채널 부재
- 네트워크 의존 → 지역·분야별 기회 불균등 심화
- 경력 활용 부재 → 사회적 고립 → 우울·건강 악화 악순환

### 수요자 — 단기 TF가 필요한 스타트업·중소기업·공공기관

스타트업이나 중소기업이 신사업 TF를 꾸릴 때 경험 많은 시니어를 3~6개월만 구하는 방법이 마땅치 않습니다. 헤드헌팅은 연봉의 20~30% 수수료가 들고, 직접 섭외는 네트워크 한계에 막힙니다.

- 검증된 시니어 전문가를 단기로 구하는 채널 없음
- 헤드헌팅 수수료 연봉의 20~30% — 단기 TF에 구조적으로 부적합
- 계약·세무 처리 번거로움으로 개인 섭외 포기
- 전문성 조달 실패 → 프로젝트 지연·실패율 상승

### 기존 해결책의 한계

| 기존 수단 | 한계 |
|-----------|------|
| 사람인·잡코리아 | 장기 정규직 중심, 시니어 단기직 카테고리 부재 |
| 크몽·숨고 | 젊은 디지털 직군 위주, 고경력 오프라인 전문가 활용 어려움 |
| 정부 신중년 사업 | 오프라인 창구 중심, AI 매칭 없음, 기업 수요와 연결 미흡 |
| 헤드헌팅 | 비용 과다, 단기 TF에 부적합한 구조 |
| **시니어링크** | **AI 정밀 매칭 + 단기 TF 특화 + 계약·정산 일체화** |

### 해결 방식

AI가 프로젝트의 성격·기간·필요 분야를 분석하여 최적의 시니어 전문가 조합을 자동 추천합니다. 계약서 자동 생성과 에스크로 정산까지 앱 안에서 해결하여, 양측 모두 절차 부담 없이 협업을 시작할 수 있습니다. 경험은 재순환되고, 양측 모두 이깁니다.

## Q4. 아이디어를 어떻게 실현하고 싶으신가요?

현재 예비창업자로서 총 15개월의 단계별 계획으로 실현합니다.

### 전체 타임라인

```
Phase 0  시장 검증   M1 ~ M2    (2개월)
Phase 1  MVP 개발    M3 ~ M8    (6개월)
Phase 2  베타 테스트 M9 ~ M11   (3개월)
Phase 3  정식 출시   M12 ~ M15  (4개월)
```

### Phase 0 — 시장 검증 (M1~M2, 2개월)

**목표**: 수요 실증 및 핵심 가설 검증

제가 직접 퇴직 동료 30명, 중소기업 대표 20명을 인터뷰합니다. 대기업·공기업 퇴직자 네트워크를 초기 전문가 풀로 활용할 수 있다는 점이 출발점의 강점입니다.

- 퇴직 전문가 30명 이상 심층 인터뷰 (경력 활용 니즈, 디지털 수용성)
- 스타트업·중소기업 20개사 인터뷰 (TF 구성 빈도, 지불 의향)
- 국내외 유사 플랫폼 벤치마킹 (탤런트뱅크, 크몽, 美 YourEncore)
- 플랫폼 노동 관련 법률 검토 및 코어팀 구성

**성공 지표**: 사용 의향 70% 이상, 파트너 기업 3개사 확보

### Phase 1 — MVP 개발 (M3~M8, 6개월)

**목표**: 핵심 기능을 담은 작동 가능한 앱 완성

개발자·디자이너를 영입해 핵심 기능 앱을 완성합니다. 시니어 사용자를 고려한 큰 글씨·간단한 UX와 카카오톡 병행 온보딩을 적용합니다.

| 기간 | 개발 내용 |
|------|-----------|
| M3~M4 | 회원가입·프로필 (전문가/기업 구분), 경력 태그 시스템 |
| M5~M6 | AI 매칭 알고리즘 v1 (스킬 키워드·지역·가용 기간 기반) |
| M7 | TF 구성 요청·제안·수락 플로우, 채팅 기능 |
| M8 | 계약서 자동 생성, 에스크로 정산, QA·버그 수정 |

**기술 스택**: React Native(iOS·Android), Node.js + PostgreSQL, 토스페이먼츠 정산 연동

### Phase 2 — 베타 테스트 (M9~M11, 3개월)

**목표**: 실사용 데이터 수집, 매칭 품질 검증

제 퇴직 동료 네트워크에서 전문가 200명을 먼저 모집하고, 중소기업 50개사와 실제 매칭을 진행합니다.

- 클로즈드 베타: 전문가 200명, 기업 50개사
- 매칭 수락률·완료율·양측 NPS 측정
- 베타 데이터 기반 AI 모델 재학습 및 고도화
- 분쟁 조정 프로세스, CS 응대 기준 수립

**성공 지표**: 매칭 완료 100건, 프로젝트 완료율 75%+, NPS 40점+

### Phase 3 — 정식 출시 (M12~M15, 4개월)

**목표**: 유료화 시작, 회원 확장, 수익 구조 안정화

- Google Play·App Store 정식 등록
- 기업 측 매칭 수수료(10~15%), 전문가 프리미엄 구독으로 수익화
- 중소기업진흥공단·창업진흥원 B2B 파트너십 협의
- 시니어 커뮤니티·경총·동창회 네트워크 활용 마케팅

**1차 목표**: 전문가 회원 5,000명, 기업 회원 500개사, 월 매칭 200건

### 필요 자원

**인력**

| 역할 | 인원 | 비고 |
|------|------|------|
| 대표/기획 (본인) | 1명 | 전략·파트너십·초기 네트워크 총괄 |
| CTO / 백엔드 | 1명 | AI 매칭 알고리즘 포함 |
| 앱 개발 (프론트) | 1명 | React Native |
| UX 디자이너 | 1명 (파트타임 가능) | 시니어 접근성 중심 설계 |
| 운영·CS | 1명 | 커뮤니티 관리, 고령자 온보딩 지원 |

**예산 (15개월 추정)**

| 항목 | 금액 |
|------|------|
| 인건비 (5인 × 15개월) | 약 1.5억~2억 원 |
| 서버·인프라 (AWS/GCP) | 약 600~1,000만 원 |
| 마케팅·PR | 약 2,000~3,000만 원 |
| 법률·회계·기타 | 약 500~1,000만 원 |
| **총계** | **약 2억~2.5억 원** |

K-Startup·중기부 TIPS 등 정부 지원금을 적극 활용하여 자부담을 최소화할 계획입니다.

### 위험 요소 및 대응

| 위험 | 대응 방안 |
|------|-----------|
